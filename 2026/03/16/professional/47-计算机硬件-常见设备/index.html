计算机硬件-常见设备

发表于 2026-03-16 更新于 2026-03-20 分类于 专业知识 ， 计算机硬件

计算机I/O设备主要包含键盘、鼠标、扫描仪等输入设备，显示器、打印机等输出设备，通过USB等接口与主机相连，不同设备按性能与用途适配不同场景。

计算机输入输出设备核心知识点梳理

一、I/O设备接口

1. 按数据传输方式分类

  • 串行接口：一次只传输1位，代表接口：USB、IEEE1394、SATA
  • 并行接口：一次传输8/16/32位，代表接口：IDE、EPP、SCSI

2. 按设备连接方式分类

  • 总线式：可连接多个设备，多设备共享，如USB
  • 独占式：只能连接1个设备，如VGA

3. USB接口核心特点

  • 通用串行总线式接口，支持热插拔、即插即用
  • 可连接最多127个设备，串行传输，可向外提供+5V、100~500mA供电
  • 典型应用：鼠标、键盘、移动硬盘、U盘、打印机等

二、常用输入设备

1. 键盘

  • 主要接口：AT接口（早期）、PS/2接口、USB接口、无线接口
  • 类型对比：
    | 类型 | 核心特点 |
    |———————|—————————————————-|
    | 电容式键盘 | 击键声音小、无磨损、寿命长、手感好 |
    | 机械式键盘 | 节奏感强、长期使用手感不变、有机械磨损 |
    | 塑料薄膜键盘 | 价格低、低噪音、长期使用手感会变化 |
    | 无接点静电电容键盘 | 无磨损、密封性好，市场主流 |

2. 鼠标器

  • 工作原理：移动转换为脉冲信号，控制光标运动
  • 核心指标：分辨率dpi（每英寸移动对应屏幕像素数，分辨率越高定位越准）
  • 分类：
    • 有线鼠标：RS-232串行口、PS/2接口、USB接口
    • 无线鼠标
    • 光机式/机械式/光电鼠标：光电鼠标速度快、精度高、分辨率可达800dpi，无机械磨损

3. 其他指点设备

  • 触摸屏：适用于服务性公共场所
  • 操纵杆：常用于游戏控制

4. 扫描仪

  • 作用：图像输入（未经OCR识别时内容不可编辑）
  • 性能指标：分辨率(dpi)、灰度、色彩位数、最大扫描幅面
  • 接口：SCSI、USB、IEEE-1394等
  • 核心原理：CCD（电荷耦合器件）完成光电转换，产生电流并输出

5. 数码相机

  • 核心性能：成像芯片（CCD/CMOS，像素越多图像质量越好）、像素与分辨率、存储器容量
  • 典型规格：约200万像素可满足一般应用，500万像素最高分辨率可达2592×1944

三、显示系统

1. 显示器

  • 常见类型：阴极射线管显示器(CRT)、液晶显示器(LCD/LED)
  • 核心性能指标：
    1. 屏幕尺寸：对角线长度（如17英寸）
    2. 分辨率：整屏可显示像素个数（如1024×1024），分辨率越高图像越清晰
    3. 刷新速率：每秒更新次数，一般≥60Hz，越高图像越稳定
    4. 颜色数：由R/G/B基色位数决定，2²⁴种颜色为真彩色

2. 显示控制器（显卡）

  • 作用：主机与显示器之间的“桥梁”
  • 核心组成：
    • 绘图处理器(GPU)：显卡核心，相当于“CPU”
    • 显示存储器(VRAM)：显卡“内存”，暂存显示芯片数据，物理上独立于PC内存
    • 接口电路
  • 常见输出端口：D-Sub(VGA)、DVI、HDMI、VIVO等
  • VRAM容量计算：屏幕分辨率 × 每个像素色彩深度（如1024×768×32bit ≈ 32MB）

四、打印机

三类打印机核心对比

类型 性质 耗材 打印质量 工作噪声 臭氧 打印速度 平均成本 定位 工作原理
针式打印机 击打式 色带 差 最大 无 慢 中 商业打印（存折/票据） 钢针打印
激光打印机 非击打式 碳粉 最好 小 有 最快 最低（初始成本高） 政府/单位 激光成像+静电吸附
喷墨打印机 非击打式 墨水 好 低 无 中 高（墨水成本高） 家庭 墨水喷射成图

关键补充

  • 激光打印机：是激光技术与复印技术的结合，多层套打是突出特点
  • 喷墨打印机：关键技术为喷头（易堵），接口多为并口/USB/SCSI（高速激光打印机）

五、高频考点速记

  1. USB：总线式、热插拔、最多127个设备、+5V供电
  2. 鼠标分辨率dpi：越高定位越精准
  3. 显示器分辨率：像素总数决定清晰度，真彩色为2²⁴种
  4. VRAM：显卡独立内存，容量=分辨率×色彩深度
  5. 打印机选型：商业票据选针式，办公文档选激光，家庭照片选喷墨
