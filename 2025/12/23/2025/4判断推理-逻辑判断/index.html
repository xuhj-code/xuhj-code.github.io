判断推理-逻辑判断

发表于 2025-12-23 分类于 判断推理 ， 逻辑判断

逻辑判断 · 总览速览表

模块 子模块 一句话核心定位 解题抓手 / 易错点
必然性推理 直言命题 结论真假由句式结构决定 矛盾关系、推出关系（所有→某个→有些）
联言命题 多条件同时成立才为真 转折词（但/却）= 且；有假即假
选言命题 至少一真或仅一真 区分相容选言与不相容选言
假言命题 条件成立必然推出结论 肯前肯后、否后否前；矛盾是 p 且非 q
综合推理 多规则叠加的必然结论题 数量限制、二难推理、假设法、连锁推理
朴素逻辑 无公式的生活化推理 确定性信息优先；列表/连线/画图
加强与削弱型 加强、削弱基础 判断论证是否更有说服力 补因果 vs 断因果；排他因 vs 引他因
因果论 因果是否真实成立 因果倒置、共同原因、样本偏差
其他常考论证 非典型论证结构 对策有效性、类比是否合理
论证细分题型 前提型 结论成立所必需条件 找“缺了就不成立”的点
解释型 现象为何会发生 解释题干而非证明结论
结论型 从事实中能推出什么 不能超出题干信息
评价型 论证是否合理、充分 看推理是否严密、是否偷换概念

使用顺序建议：
先判模块 → 再对照子模块 → 最后套对应解题套路

逻辑判断 · 知识点小结速览表

知识点 核心内容 考试重点 / 易混淆点 难度系数
直言命题基础 六种句式结构：所有A是B / 非B；有些A是B / 非B；某个A是B / 非B 句式识别与变形（“都”=“所有”，“至少一个”=“有些”） ★★☆☆☆
矛盾关系 找矛盾两种方式：①句前加“并非” ②变形规则（所有↔有些，是↔非） 矛盾必一真一假；某个A是B ↔ 某个A非B ★★★☆☆
反对关系 上反对：所有A是B vs 所有A非B（必有一假）
下反对：有些A是B vs 有些A非B（必有一真）		 真假话题优先找矛盾，无矛盾再用反对关系 ★★★★☆
推出关系 推理链条：所有 → 某个 → 有些（不可逆） 冲突关系：某个A是B → 所有A非B假；注意单项判断特性（不能互推） ★★★☆☆
复杂句式拓展 “不都是”= 有些非
“没有一个A是B”= 所有A非B
“没有一个A不是B”= 所有A是B		 并非开头的句子：去“并非”+变矛盾 ★★☆☆☆
联言命题 标准形式 p 且 q（同时为真） 提示词：既…又 / 不但…而且 / 但是；推理规则：全真才真，有假则假；矛盾形式：非p 或 非q ★★☆☆☆
真假话解题法 三步走：①找矛盾 / 反对 ②绕开看其余 ③回代判断 秒杀技巧：三句话 + 反对关系时，反着第三句变所有 ★★★★★
联言命题 表示多种情况同时为真（且关系） 转折词（但 / 却）也表示且关系；全真才真，有假则假；矛盾形式：非p 或 非q ★★☆☆☆
相容选言命题 至少有一真（或关系） “或者……或者……两者只需其一”实为不相容选言，易混 ★★★☆☆
不相容选言命题 有且只有一真（要么……要么……） 与相容选言命题的真值表区别 ★★☆☆☆
假言命题 推理规则：肯前推肯后；否后推否前 否前、肯后无确定结论（带“可能”除外） ★★★★☆
假言命题的矛盾 p → q 的矛盾是 p 且 非q 必须严格用“且”，不能用“或” ★★★☆☆
真假话问题解法 三步走：①找矛盾 → ②绕矛盾 → ③回矛盾 优先带回非假言命题（且关系）判断 ★★★★☆
传递性推理 相同概念可串联形成推理链 需注意逆否命题转换 ★★★★☆
除非 / 否则不结构 后推前 “否则不 + 否则”= 加双重否定；与“只有……才……”易混淆 ★★★☆☆
知识点 核心内容 考试重点 / 易混淆点 难度系数
假言命题（逻辑等价） p → q 等价于 ¬p ∨ q 区分“推出关系成立”与“命题为真”的区别；矛盾关系为 p ∧ ¬q ★★★☆☆
二难推理 结构：A → B 且 ¬A → B ⇒ B 恒真 识别题干中的矛盾前提（A / ¬A） ★★★★☆
综合推理题型判定 题干含假言命题 + 选项为事实 问法“知真求真”；未提真假时默认题干为真 ★★★☆☆
数量限定解题法 三选二 ⇒ 任意二至少选一 七选四等宏观数量与局部条件的联动 ★★★★☆
连锁推理 推理链从头或尾切入假设（如 王→李→老→宋→东） 假设暴露后需验证是否与总数冲突 ★★★★☆
假设法（分类讨论） 分情况验证（如 投资甲 / 不投资甲） 冲突则反证；两种无冲突情况需综合共性（如“橡皮树恒选”） ★★★★★
真假话问题拓展 假言无矛盾时，将 p → q 转化为 ¬p ∨ q 判断 倒数第二题“非甲或乙”的转换应用 ★★★★☆
且 / 或关系优先级 且、或与箭头共存 优先按箭头断句；“去四川 → 去北京且上海”的否后技巧 ★★★☆☆
知识点 核心内容 考试重点 / 易混淆点 难度系数
朴素逻辑 日常生活信息推理，无公式化表达（如“如果那么”），需通过突破口（确定性 / 关联性）和辅助方法（列表 / 连线 / 画图 / 假设）解题 区分“确定性信息”与“关联性信息”的优先级；连线法仅适用于三类元素一一匹配场景 ★★★★☆
综合推理 题干含假言命题，选项为事实，通过确定信息 / 数量限制 / 二难推理 / 假设法解题 数量限制的灵活应用（如“三选二”）；二难推理中“a推b，非a推b ⇒ b恒真” ★★★★☆
真假话问题 矛盾关系（如 p 推 q vs p 且非 q）、反对关系（所有是 vs 所有非）、套路题型（冠军模型 / 半真半假） 套路题型优先用（如“出现次数最多信息为真”）；无套路时需假设法 ★★★★★
假言命题匹配 翻译规则（前推后 / 后推前）、肯前肯后 / 否后否前有效推理 警惕否前肯后的无效推理；重复概念需串联（如 p 推 q，q 推 r ⇒ p 推 r） ★★★★☆
知真求假 / 知假求真 题干真假与选项矛盾关系（如“题干为假 ⇒ 选其矛盾命题”） 多句话串联时优先关注头尾矛盾（如“甲推非丁”的矛盾是“甲且丁”） ★★★☆☆
直言命题真假 矛盾关系（所有是 vs 有些非）、推出关系（所有是 ⇒ 某个是 ⇒ 有些是） “并非所有”需转化为“有些非”；“有些”无法推出“某个” ★★☆☆☆
