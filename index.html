多媒体技术-考前狂背清单

发表于 2026-03-17 分类于 专业知识 ， 多媒体技术

多媒体技术 · 考前狂背清单（30行精简版）

一、五类媒体（必背）

  1. 感觉媒体：文字、声音、图像、视频
  2. 表示媒体：编码、ASCII、Unicode、音视频编码
  3. 显示媒体：输入输出设备（键盘、显示器、麦克风）
  4. 存储媒体：U盘、硬盘、光盘、内存
  5. 传输媒体：光纤、双绞线、无线电波

二、汉字编码

  1. GB2312：简体汉字
  2. GBK：兼容GB2312，含繁体
  3. GB18030：最全，兼容前面所有
  4. Unicode：国际统一编码
  5. BIG5：台湾繁体编码

三、汉字输入编码

  1. 数字编码：区位码、电报码
  2. 字音编码：拼音（易學、重碼多）
  3. 字形编码：五笔（快、难學）
  4. 形音编码：字形+字音

四、音频

  1. 人耳：20Hz～20kHz；话音：300～3400Hz
  2. 音频数字化：采样 → 量化 → 编码
  3. 奈奎斯特：采样频率 ≥ 最高频率2倍
  4. 音频容量 = 采样率×位数×声道数×时间 ÷8
  5. WAV无压缩；MP3有损；FLAC无损；MIDI是指令

五、图像

  1. 彩色三要素：亮度、色调、饱和度
  2. 图像容量 = 分辨率×像素深度 ÷8
  3. 颜色数 = 2的像素深度次方
  4. 位图：放大会失真
  5. 矢量图：放大不失真（logo、线条）

六、压缩与格式

  1. 无损压缩：可完全还原
  2. 有损压缩：不可完全还原
  3. 静态图压缩标准：JPEG
  4. 动态图/视频：MPEG
  5. BMP无压缩；JPG照片；PNG透明；GIF动图

七、视频

  1. 视频 = 图像序列 + 音频

多媒体技术 · 考前押题10道（含答案）

1. 下列属于感觉媒体的是（）

A. 光纤 B. ASCII C. 声音 D. 显示器
答案：C

2. 为传输、处理感觉媒体而设计的编码属于（）

A. 显示媒体 B. 表示媒体 C. 存储媒体 D. 传输媒体
答案：B

3. 汉字编码从早到晚的顺序是（）

A. GBK→GB2312→GB18030
B. GB2312→GBK→GB18030
C. GB18030→GBK→GB2312
D. GB2312→GB18030→GBK
答案：B

4. 拼音输入法属于汉字编码中的（）

A. 数字编码 B. 字音编码 C. 字形编码 D. 形音编码
答案：B

5. 音频数字化的正确步骤是（）

A. 采样→量化→编码
B. 量化→采样→编码
C. 编码→采样→量化
D. 采样→编码→量化
答案：A

6. 根据奈奎斯特，要不失真采集10kHz声音，采样频率至少为（）

A. 10kHz B. 20kHz C. 5kHz D. 40kHz
答案：B

7. 一幅 1024×768、24位真彩色图像，数据量约为（）

A. 2.25MB B. 3MB C. 1.5MB D. 6MB
答案：A

8. 放大后不会失真的是（）

A. 位图 B. 矢量图 C. 照片 D. BMP图像
答案：B

9. JPEG是哪类媒体的压缩标准（）

A. 音频 B. 静态图像 C. 视频 D. 文本
答案：B

10. 下列属于无损压缩的是（）

A. MP3 B. JPEG C. PNG D. MPEG
答案：C

多媒体技术-备考速记清单

发表于 2026-03-17 分类于 专业知识 ， 多媒体技术

多媒体技术 · 备考速记清单（背诵版）

一、5 类媒体（必考）

  1. 感觉媒体：人直接感觉到的——文字、声音、图像、动画、视频
  2. 表示媒体：计算机编码——ASCII、Unicode、图像编码、音频编码
  3. 显示媒体：输入输出设备——键盘、显示器、麦克风、打印机
  4. 存储媒体：存数据的介质——U盘、硬盘、光盘、内存
  5. 传输媒体：传数据的线路——光纤、双绞线、无线电波

口诀：感觉是内容、表示是编码、显示是设备、存储是介质、传输是线路

二、汉字编码（必背）

1. 编码演进

  • GB2312：简体中文，6763 个汉字
  • GBK：兼容 GB2312，含繁体
  • GB18030：最完整，兼容 GB2312、GBK
  • Unicode：国际通用编码
  • BIG5：台湾繁体编码

2. 汉字输入编码 4 类

  1. 数字编码：区位码、电报码（难记）
  2. 字音编码：拼音（易学、重码多）
  3. 字形编码：五笔（重码少、难学）
  4. 形音编码：字形+字音

3. 文本输入方式

  • 人工：键盘、手写、语音
  • 自动：OCR（印刷体）、手写体识别

三、音频（计算+概念）

  1. 人耳听觉：20Hz～20kHz
  2. 话音：300～3400Hz
  3. 音频数字化三步：采样 → 量化 → 编码
  4. 奈奎斯特：采样频率 ≥ 声音最高频率 2 倍

音频数据量公式（必考计算）

[
数据量(Byte) = \frac{采样频率\times 量化位数\times 声道数\times 时间(s)}{8}
]

音频格式

  • WAV：无压缩，体积大
  • MP3：有损压缩
  • FLAC：无损压缩
  • MIDI：不是声音，是音乐指令，极小

四、图像（计算+分类）

  1. 彩色三要素：亮度、色调、饱和度
  2. 图像数字化：扫描 → 分色（RGB）→ 取样 → 量化
  3. 像素深度：一个像素用多少位
    • 颜色数 = (2^{像素深度})

图像容量公式

[
数据量(Byte) = \frac{分辨率\times 像素深度}{8}
]

压缩

  • 无损：可完全还原（文字、图标）
  • 有损：不能还原（照片、视频）
  • 静态图：JPEG
  • 动态图：MPEG

图像格式

  • BMP：无压缩
  • JPG：有损，照片用
  • PNG：无损，透明，网页用
  • GIF：256色，动图

位图 vs 矢量图

  • 位图：放大会糊，适合照片
  • 矢量图：放大不失真，适合logo、线条图

五、视频（一句话考点）

  1. 视频 = 序列图像 + 音频
  2. 视频容量 = 每帧图像大小 × 帧率 × 时间 + 音频
  3. 标准：MPEG

六、超高频考点一句话

  • 媒体五类：感觉、表示、显示、存储、传输
  • 汉字编码：GB2312 → GBK → GB18030
  • 音频数字化：采样、量化、编码
  • 采样 ≥ 最高频率 2倍
  • 图像：分辨率×位深÷8 = 容量
  • 位图失真，矢量图不失真
  • 压缩：无损可还原，有损不还原

多媒体技术-音频与图像基础

发表于 2026-03-16 更新于 2026-03-21 分类于 专业知识 ， 多媒体技术

本文接续上篇，整理多媒体技术中音频、图像的核心概念、计算与格式，适合备考与技术入门查阅。

📝 核心公式

  • 文本：1汉字 ≈ 2字节（UTF-8）
  • 图像：大小 = 分辨率 × 色深 ÷ 8
  • 音频：大小 = 码率(kbps) × 时长(秒) ÷ 8
  • 视频：大小 = 码率(Mbps) × 时长(秒) ÷ 8

📊 典型大小对比

格式 示例场景 近似大小 量级
TXT 100万字中文文本 ≈ 2 MB 文本级
BMP 1024×768，24位色位图 ≈ 2.25 MB 图像级
JPG 1024×768，中等质量图片 50 KB~200 KB 图像级
PNG 1024×768，32位色透明图片 200 KB~500 KB 图像级
MP3 10分钟，128kbps音频 ≈ 93.75 MB 音频级
WAV 10分钟，44.1kHz/16位立体声 ≈ 100.8 MB 音频级
MPEG 10分钟，1Mbps视频 ≈ 75 MB 视频级
MP4 10分钟，2Mbps视频 ≈ 150 MB 视频级

⚡ 快速判断技巧

  1. 体积排序：文本 < 压缩图片 < 无损图片 < 音频 < 视频
  2. 同类型对比：
    • 图片：BMP > PNG > JPG
    • 音频：WAV > MP3
    • 视频：分辨率/码率越高，体积越大
  3. 高频考点经验值：
    • 100万字TXT ≈ 2MB
    • 1分钟128kbps MP3 ≈ 9.4MB
    • 1分钟1Mbps MPEG ≈ 7.5MB

✅ 真题答案

下列文件中数据量最小的是：
A. 100万字TXT B. 1024×768×24位BMP C. 10分钟MP3 D. 10分钟MPEG
答案：A

计算机操作系统-练习题-总结

发表于 2026-03-16 更新于 2026-03-22 分类于 专业知识 ， 操作系统

操作系统·错题一页速记卡

一、进程与线程（高频错）

  1. 进程存在唯一标志：PCB（进程控制块），不是 PID
  2. 同步=按序执行；调度=分配CPU；互斥=临界资源排他访问
  3. 无饥饿算法：时间片轮转；短作业优先会饥饿
  4. 时间片轮转≠优先级调度，用于分时系统
  5. 程序静态、进程动态；一个程序可对应多个进程
  6. 只要有一个线程运行，进程状态=运行态

二、死锁与资源

  1. 死锁不发生于：CPU（可抢占）
  2. 死锁最小资源公式：进程数×(需求-1)+1
  3. 银行家算法=避免死锁，不是预防死锁
  4. 预防死锁：破坏请求保持、不剥夺、环路等待

三、内存管理

  1. 虚拟内存 = 内存 + 硬盘交换区
  2. 时间片大小由响应时间要求决定
  3. 缓冲技术解决：高速 CPU 与低速 I/O 速度差

四、文件系统

  1. 文件系统核心目的：按名存取
  2. 文件逻辑结构：有结构/无结构（流式）
  3. Windows 非法字符：* ? \ / : " < > |
  4. 虚拟内存页面文件：pagefile.sys

五、操作系统基础

  1. 最接近硬件：BIOS/UEFI
  2. 单用户 OS：MS-DOS；Windows Server 是多用户
  3. OS 四大功能：进程、内存、设备、文件管理
  4. OS 存硬盘，运行时加载进内存

六、硬盘与 I/O

  1. 硬盘核心指标：容量、转速、平均寻道时间
  2. I/O 控制器管理设备全过程

七、软件著作权（易错）

  • 学习研究反向工程：不侵权
  • 盗版商用、一机一装多机、无法证明合法来源：均侵权
