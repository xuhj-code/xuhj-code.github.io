图形推理-位置与样式

发表于 2025-12-23 分类于 判断推理 ， 图形推理 ， 位置与样式

位置规律 · 核心速览

位置规律 相邻比较 旋转翻转
平移方向 + 步数，16 宫格内外分开 找不动元素，其余只换位置 顺逆时针角度 + 翻转轴
九宫格规律 16 宫格 25 宫格与蜂窝
等差 / 恒定 / 周期，横向优先 内外分离，重点内部递增 斜向整体运动，大面积定位
非宫格难点 解题技巧 心理建设
隐藏线条、拆分运动 大面积 → 个数 → 对比 难题跳过，70% 即高分

样式规律 · 核心速览

样式规律 黑白运算 标记考点
部分相同元素做运算，加减 / 求同 / 求异（高频求异） 外框相同看黑块变化，相同区域秒杀优先 点 / 线 / 角 / 面，关注标记位置与关系
动态样式 拆分思维 金字塔运算
先旋转再运算，一三同转二 / 二三同转一 不同区域不同规则，上下或内外分开 下层两元素决定上层一元素

属性规律 · 核心速览

对称性 曲直性 开闭性
看对称轴方向与数量，区分轴 / 中心 / 轴+中心 用 1/2、2/3 比例定位，判断是否含曲线 生活化优先：全开 / 半开 / 全闭（点连接也算开）
图形间关系 黑白块专项 最新趋势
简单看相交点线面，复杂找共同部分 + 差异 顺序：对称性 ＞ 位置规律 ＞ 黑白运算 不看整体，看未相交部分或相交线位置

必然性推理｜考场判断总览

发表于 2025-12-20 更新于 2025-12-23 分类于 判断推理

一、什么是必然性推理（考场定义）

结论 一定成立 / 一定不成立
不接受「可能」「大概率」

二、必然性推理三大入口（先找这个）

1️⃣ 形式逻辑入口

  • 如果…那么…
  • 只有…才…
  • 除非…

👉 优先想：逆否命题

2️⃣ 条件组合入口

  • 多个条件同时出现
  • 至少 / 至多 / 恰有

👉 优先想：列关系，不算数

3️⃣ 排除唯一入口

  • 只有一个选项不矛盾
  • 其余均违反已知条件

👉 优先：排除法

三、必然性推理常用技巧

  • 逆否命题
  • 设反例
  • 代入检验

四、考场提醒（只记这个）

  • ❌ 不讨论概率
  • ❌ 不补充常识
  • ✅ 只看题干给的条件

资料分析速算技巧

发表于 2025-11-27 更新于 2026-03-17 分类于 资料分析

0️⃣ 小技巧汇总（便于速记）

  • 加法：高位叠加 → 精算 一加到底；估算 两加一估
  • 减法：划线减法 → 减少借位
  • 乘法：高位乘法 / 拆分法（接近整十）/ 口诀法（不接近整十）
  • 特殊数：×1.5、×1.1（错位相加）、×0.9（错位相减）
  • 百化分：按图片分类（等差数列 / 母子互换 / 加和20），左—右互换便于记忆
