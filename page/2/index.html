增长量 · 核心速览

增长量概念 增长量公式 累计类材料
增长量=现期-基期，带单位 vs 百分号 现期-基期 / 基期×增长率 / 现期/(1+r)×r 季度=全年-前三季度；月度=当月累计-上月累计
特定增长量 年均增长量 现期量+增长率求增长量
现期-基期>X 或 基期+X<现期 江苏/国家规划 n=年数；前推基期 r>0 用 n+1；r<0 用 n-1，误差分析
混合增长量 增长量倍数 百分点增长量
总体=各部分和，正负抵消 增长率差≤10%可约简 每增长 n 个百分点=基期×n%
增长量比较 增长贡献率 差值变化判断
柱状/折线图看高度差，正负分组比较 部分增量/整体增量，分母相同看分子 前者增量>后者增量→差值扩大，反则缩小

现期量 · 核心速览

现期量公式 限期追赶 小追大（追得上/追不上）
增量保持用加，增率保持用乘 代入法快速判断年份 补差距 / 加差距 → 年限计算
大追小（倍数目标） 名义 vs 实际 基期价计算
现期量=基期量×(1+增长率)^n 名义=实际+CPI+交叉项，名义>实际→物价涨 用实际增长率反推基期量
利滚利思想 核心策略 快速判断
拆分本金与利息，大额精确，小额估算 比例关系快速判断 先抓公式，关键差距/倍数

基期是对比参照期，与现期相对 基期=现期−增长量 / 现期÷(1+增长率) 同比看年份，环比看相邻最小时间

基期和差 累计值 进出口
整体/比重求基期，差值需反推 季度=累计−上期累计 顺差=出口−进口

图形推理-数量规律

发表于 2025-12-23 分类于 判断推理 ， 图形推理 ， 数量规律

数量规律 ·面·线·点· 核心速览

对称轴优先 面的考法 线的考法
对称轴是第一优先级，黑白块先看轴 白色封闭才算面，缺口不算 拐弯算多条线，先分直 / 曲
轴对称 / 中心对称直接秒 面数可横向 / 纵向相加 两种线常考相加 / 倍数
对称成立，其它规律不用看 最大面 / 相同面是高频 平行线按“同方向一组”
点的考法 曲线快速分流 相邻比较
只数交点，端点不算 圆 → 切点；非圆 → 曲直交点 有相同就比，不同是突破口
多端点先想笔画 部分曲线 ≠ 全曲线 动的是变量，不动是关键
点常配合线一起考 曲线强于面特征 比位置 / 比数量 / 比存在
优先级总纲
对称轴 ＞ 线 ＞ 面 ＞ 点，先抓最显眼特征 不要一上来就数面
曲线特征 ＞ 面特征 复杂题先排明显错误选项
看不出就回到“对称 + 相邻比较”

数量规律 ·笔画·元素·直角· 总览速览表

笔画数 元素考点 部分数
去外框、找基点，生活化图形优先 独立元素，算种类与数量 连在一起算一部分，对角相连
直角考点 黑白块专项 生活化图形
十字 / 矩形 / 直角三角形，白算黑不算 先位置，再黑白，再部分数 开闭性、面数、部分数、笔画数
换算公式 解题流程 考场策略
一加三=两倍图二（眼镜法则） 对称 → 数量 → 图形关系 难题跳过，选项对比

图形推理-位置与样式

发表于 2025-12-23 分类于 判断推理 ， 图形推理 ， 位置与样式

位置规律 · 核心速览

位置规律 相邻比较 旋转翻转
平移方向 + 步数，16 宫格内外分开 找不动元素，其余只换位置 顺逆时针角度 + 翻转轴
九宫格规律 16 宫格 25 宫格与蜂窝
等差 / 恒定 / 周期，横向优先 内外分离，重点内部递增 斜向整体运动，大面积定位
非宫格难点 解题技巧 心理建设
隐藏线条、拆分运动 大面积 → 个数 → 对比 难题跳过，70% 即高分

样式规律 · 核心速览

样式规律 黑白运算 标记考点
部分相同元素做运算，加减 / 求同 / 求异（高频求异） 外框相同看黑块变化，相同区域秒杀优先 点 / 线 / 角 / 面，关注标记位置与关系
动态样式 拆分思维 金字塔运算
先旋转再运算，一三同转二 / 二三同转一 不同区域不同规则，上下或内外分开 下层两元素决定上层一元素

属性规律 · 核心速览

对称性 曲直性 开闭性
看对称轴方向与数量，区分轴 / 中心 / 轴+中心 用 1/2、2/3 比例定位，判断是否含曲线 生活化优先：全开 / 半开 / 全闭（点连接也算开）
图形间关系 黑白块专项 最新趋势
简单看相交点线面，复杂找共同部分 + 差异 顺序：对称性 ＞ 位置规律 ＞ 黑白运算 不看整体，看未相交部分或相交线位置
