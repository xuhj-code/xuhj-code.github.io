多媒体信息技术-练习题-总结

发表于 2026-03-16 更新于 2026-03-22 分类于 专业知识 ， 操作系统

📝 多媒体信息技术 一页速记卡

🖼️ 图像类

  • 位图 vs 矢量图
    • 位图：像素组成，放大失真，格式：BMP/JPG/GIF/PNG
    • 矢量图：指令组成，放大不失真，格式：WMF/SWF/SVG
    • ✅ 口诀：位图失真色彩艳，矢量无损体积小
  • 图像格式
    • GIF：支持静态+透明+动画
    • JPEG：网页常用有损压缩
    • ✅ 口诀：GIF动图透背景，JPEG网页省空间
  • 分辨率与画布
    • 画布大小≠分辨率，像素总数不变则分辨率不变
    • ✅ 口诀：改画布不改像素，分辨率纹丝不动
  • 数据量计算
    • 公式：数据量(Byte) = 分辨率 × 量化位数 ÷ 8（彩色×3通道）
    • ✅ 口诀：分辨率乘位数，除以8变字节，彩色要乘3

🎧 音频类

  • MIDI vs 波形
    • MIDI：指令序列，不是波形，体积极小，可编辑
    • 波形：WAV/MP3，真实采样，体积大
    • ✅ 口诀：MIDI是指令不是声，体积小来易编辑
  • 音质影响因素
    • 只看：采样频率、量化位数、声道数
    • 存储介质不影响音质
    • ✅ 口诀：采样量化加声道，存储介质不重要
  • 音频数字化
    • 顺序：采样 → A/D转换 → 压缩 → 存储 → 解压缩 → D/A转换
    • 硬件：采样量化用A/D转换器，播放用D/A
    • ✅ 口诀：先采样再转数字，压缩存储再还原
  • 声音频段
    • <20Hz：次声；20Hz~20kHz：可听声；>20kHz：超声
    • 人耳最敏感：2kHz~4kHz
    • ✅ 口诀：次低超高听不见，2到4千最敏感

🎬 视频类

  • 压缩标准
    • VCD：MPEG-1；DVD：MPEG-2；网络视频：MPEG-4
    • 静态图：JPEG；无MPEG-3视频标准
    • ✅ 口诀：VCD用1，DVD用2，网络视频用4
  • 视频核心概念
    • 最小单位：帧
    • 流媒体：边下载边播放，无需完整文件
    • ✅ 口诀：视频最小是帧，流媒体边下边看
  • 视频质量
    • 分辨率↑、颜色位数↑、帧率↑ → 质量越好
    • ✅ 口诀：分辨率高颜色深，帧率快了画质好

🗜️ 数据压缩类

  • 有损 vs 无损
    • 无损：文本/程序，可完全恢复（哈夫曼、LZW、游程编码）
    • 有损：图像/音频/视频，不可完全恢复（预测、变换DCT编码）
    • ✅ 口诀：文本程序要无损，音视频可有损
  • JPEG编码
    • 流程：DCT变换 → 量化 → 熵编码（哈夫曼/算术编码）
    • ✅ 口诀：先变换再量化，最后熵编码
  • 压缩评价指标
    • 核心：压缩比、恢复质量、编解码速度
    • 标准化不是性能指标
    • ✅ 口诀：比质量比速度，压缩比是核心

🧩 媒体与超文本

  • 媒体分类
    • 多媒体“媒体”指：感觉媒体（声音、图像、文字）
    • 显示媒体：电信号↔感觉媒体转换（显示器、音箱）
    • ✅ 口诀：多媒体是感觉，显示媒体转信号
  • 超文本三要素
    • 节点（信息块）、链（连接）、网络（结构）
    • ✅ 口诀：节点加链成网络，超文本结构就这仨
  • 多媒体三大特性
    • 多样性、集成性、交互性
    • ✅ 口诀：多集交，多媒体

🎨 颜色模型

  • 屏幕显示：RGB
  • 打印输出：CMYK
  • 电视传输：YUV
  • RGB(0,0,0)=黑色；RGB(255,255,255)=白色
  • ✅ 口诀：屏RGB，印CMYK，电视YUV；全0是黑全白是亮

计算机硬件-练习题

发表于 2026-03-16 更新于 2026-03-22 分类于 专业知识 ， 计算机硬件

计算机软硬件错题集

科目：计算机组成原理

一、单选题（共40道）

题目1

题干：下列相对于CISC，关于RISC的说法中，不正确的是（ ）。
A.指令长度固定
B.指令种类少
C.寻址方式少
D.不适合流水线操作
你的答案：D
正确答案：D
备注：作答正确，用于巩固知识点
考点：RISC与CISC的核心特点对比
解题技巧：

  • RISC：精简指令、固定长度、寻址方式少、适合流水线
  • CISC：复杂指令、长度可变、寻址方式多、不适合流水线

题目2

题干：计算机采用分级存储体系的主要目的是为了解决（ ）的问题。
A.主存储器容量不足
B.存储器读写可靠性
C.外设访问效率
D.存储容量、成本和速度之间的矛盾
你的答案：D
正确答案：D
备注：作答正确，用于巩固知识点
考点：分级存储体系的设计目标
解题技巧：

  • 本质：用不同成本、速度的存储介质（Cache-主存-辅存）搭配，平衡容量、成本、速度三者矛盾

题目3

题干：下列哪个单位常用于度量存储器的容量？（ ）
A.Mb/s
B.GHz
C.KB
D.dpi
你的答案：C
正确答案：C
备注：作答正确，用于巩固知识点
考点：存储容量单位识别
解题技巧：

  • 存储容量：B、KB、MB、GB、TB
  • 传输速率：Mb/s、Gb/s
  • 频率：Hz、GHz
  • 分辨率：dpi

题目4

题干：内存按字节编址，地址从B4000H到DBFFFH，共有（ ）个字节。
A.80K
B.96K
C.160K
D.192K
你的答案：C
正确答案：C
备注：作答正确，用于巩固知识点
考点：内存地址范围计算
解题技巧：

  • 公式：结束地址 - 起始地址 + 1 = 存储单元数
  • 计算：DBFFFH - B4000H + 1 = 28000H = 163840字节 = 160K

题目5

题干：在CPU中，（ ）可用于传送和暂存用户数据，为ALU执行算术逻辑运算提供工作区。
A.程序计数器
B.累加寄存器
C.程序状态寄存器
D.地址寄存器
你的答案：B
正确答案：B
备注：作答正确，用于巩固知识点
考点：CPU内部寄存器功能
解题技巧：

  • 程序计数器（PC）：存下一条指令地址
  • 累加寄存器（ACC）：暂存运算数据，为ALU提供工作区
  • 程序状态寄存器（PSW）：存运算标志位
  • 地址寄存器（AR）：存内存访问地址

题目6

题干：逻辑运算“1 XOR 1”的结果是（ ）。
A.0
B.1
C.2
D.不确定
你的答案：A
正确答案：A
备注：作答正确，用于巩固知识点
考点：逻辑异或运算
解题技巧：

  • 口诀：相同为0，不同为1
  • 示例：1 XOR 1 = 0，1 XOR 0 = 1

题目7

题干：位于CPU与主存之间的高速缓冲存储器Cache用于存放部分主存数据的拷贝，主存地址与Cache地址之间的转换工作由（ ）完成。
A.硬件
B.软件
C.用户
D.程序员
你的答案：A
正确答案：A
备注：作答正确，用于巩固知识点
考点：Cache地址转换机制
解题技巧：

  • Cache是硬件层面优化，地址转换由Cache控制器（硬件）自动完成，对用户/程序员透明

题目8

题干：对于8位带符号位定点整数，补码能表示的数的范围是（ ）。
A.0至256
B.-127至127
C.0至127
D.-128至127
你的答案：C
正确答案：D
错误原因：混淆了原码和补码的表示范围，误将补码范围记为原码范围
考点：补码的表示范围
解题技巧：

  • 公式记忆：n位带符号整数补码范围为 -2^(n-1) ~ 2^(n-1)-1
  • 8位特例：-128（10000000）~ 127（01111111），补码中0只有一种表示，可多表示一个负数

题目9

题干：基本ASCII字符集共包含多少个字符？（ ）
A.64
B.128
C.256
D.512
你的答案：B
正确答案：B
备注：作答正确，用于巩固知识点
考点：ASCII字符集大小
解题技巧：

  • 基本ASCII：0~127共128个字符（7位编码）
  • 扩展ASCII：0~255共256个字符（8位编码）

题目10

题干：按操作系统的分类，UNIX操作系统是（ ）。
A.批处理操作系统
B.实时操作系统
C.分时操作系统
D.单用户操作系统
你的答案：A
正确答案：C
错误原因：混淆了操作系统分类，UNIX是典型的分时操作系统，允许多用户同时交互使用，而非批处理系统
考点：操作系统的分类
解题技巧：记忆典型系统：分时系统（UNIX、Linux）、批处理系统（早期大型机）、实时系统（工业控制）、单用户系统（早期DOS）

题目11

题干：十进制数“-1”的8位补码表示是（ ）
A.10000001
B.11111110
C.11111111
D.00000001
你的答案：B
正确答案：C
错误原因：补码计算错误。-1的原码是10000001，反码是11111110，补码是反码+1，即11111111
考点：负数补码的计算
解题技巧：补码=原码除符号位外取反+1；或记住-1的8位补码固定为11111111

题目12

题干：目前U盘和SSD固态硬盘主要使用的存储技术是（ ）
A.DRAM
B.SRAM
C.闪存
D.磁盘
你的答案：C
正确答案：C
备注：作答正确，用于巩固知识点
考点：存储技术的应用
解题技巧：闪存：U盘/SSD/SD卡；DRAM：主存；SRAM：Cache；磁盘：机械硬盘

题目13

题干：电子计算机的性能可以用很多指标来衡量，除了用其运算速度、字长等主要指标以外，还可以用下列（ ）来表示。
A.主存储器容量的大小
B.硬盘容量的大小
C.显示器的尺寸
D.计算机制造成本
你的答案：A
正确答案：A
备注：作答正确，用于巩固知识点
考点：计算机性能指标
解题技巧：核心性能指标：运算速度、字长、主存容量、Cache容量等；辅助指标：硬盘容量、外设配置等

题目14

题干：显示或打印汉字时，系统使用的是汉字的（ ）
A.机内码
B.字形码
C.输入码
D.国标码
你的答案：B
正确答案：B
备注：作答正确，用于巩固知识点
考点：汉字编码的分类
解题技巧：区分各类编码：输入码（输入法用）、机内码（存储用）、字形码（显示/打印用）、国标码（编码标准）

题目15

题干：字符比较大小实际是比较它们的ASCII码值，下列正确的比较是（ ）
A.“A”比“B”大
B.“H”比“h”小
C.“F”比“D”小
D.“9”比“D”大
你的答案：B
正确答案：B
备注：作答正确，用于巩固知识点
考点：ASCII码值的大小比较
解题技巧：记忆顺序：数字(0-9) < 大写字母(A-Z) < 小写字母(a-z)，同类型按字母/数字顺序递增

题目16

题干：指令寄存器的位数取决于（ ）。
A.存储器的容量
B.指令字长
C.数据总线的宽度
D.地址总线的宽度
你的答案：D
正确答案：B
错误原因：混淆了指令寄存器与地址寄存器的作用。指令寄存器用于存放当前指令，位数由指令字长决定；地址寄存器位数由地址总线宽度决定
考点：CPU寄存器与指令字长的关系
解题技巧：指令寄存器位数=指令字长；地址寄存器位数=地址总线宽度；数据寄存器位数=数据总线宽度

题目17

题干：下列各种进制数的表示中，非法的是（ ）
A.1000100111
B.320D
C.FEDCH
D.418O
你的答案：A
正确答案：D
错误原因：八进制数（后缀O）只能包含0-7，418O中出现了数字8，非法；320D是合法十进制，FEDCH是合法十六进制，1000100111是合法二进制
考点：不同进制数的表示规则
解题技巧：二进制(0-1)、八进制(0-7)、十进制(0-9)、十六进制(0-9,A-F)，后缀分别为B/O/D/H

题目18

题干：若某进制的运算中5X9=39，则在该进制下4X7的结果是（ ）
A.11
B.14
C.24
D.28
你的答案：C
正确答案：C
备注：作答正确，用于巩固知识点
考点：进制转换与运算
解题技巧：设未知进制为r，将等式按权展开解方程求r，再进行运算转换

题目19

题干：下列接口中，不能支持热插拔的是（ ）
A.USB
B.SATA
C.PS/2
D.IEEE 1394
你的答案：D
正确答案：C
错误原因：混淆了接口的热插拔特性。PS/2接口（老式键盘鼠标接口）不支持热插拔，USB、SATA、IEEE 1394均支持
考点：计算机接口的热插拔特性
解题技巧：记忆支持热插拔的接口：USB、SATA、Thunderbolt、IEEE 1394；不支持：PS/2、VGA、并口

题目20

题干：硬盘属于（ ）。
A.内部存储器
B.外部存储器
C.只读存储器
D.输出设备
你的答案：B
正确答案：B
备注：作答正确，用于巩固知识点
考点：存储设备的分类
解题技巧：内部存储器（RAM/ROM）：CPU直接访问；外部存储器（硬盘/U盘/SSD）：需通过接口访问

题目21

题干：显示器的什么指标越高，显示的图像越清晰？（ ）
A.对比度
B.对比度和亮度
C.亮度
D.分辨率
你的答案：D
正确答案：D
备注：作答正确，用于巩固知识点
考点：显示器性能指标
解题技巧：分辨率=水平像素数×垂直像素数，是决定图像清晰度的核心指标

题目22

题干：DMA(直接存储器访问)工作方式是在（ ）之间建立起直接的数据通路
A.CPU与外设
B.CPU与主存
C.主存与外设
D.外设与外设
你的答案：C
正确答案：C
备注：作答正确，用于巩固知识点
考点：DMA工作原理
解题技巧：DMA的核心是“主存-外设”直接数据传输，绕过CPU

题目23

题干：主板上的BIOS芯片通常是一种（ ）
A.DRAM
B.SRAM
C.ROM
D.Cache
你的答案：C
正确答案：C
备注：作答正确，用于巩固知识点
考点：BIOS芯片的存储类型
解题技巧：ROM/Flash ROM用于存储固件（BIOS、固件程序），RAM易失，Cache是高速缓存

题目24

题干：以下关于Cache(高速缓冲存储器)的叙述中，不正确的是（ ）。
A.Cache 的设置扩大了主存的容量
B.Cache 的内容是主存部分内容的拷贝
C.Cache 的命中率并不随其容量增大线性地提高
D.Cache 位于主存与 CPU 之间
你的答案：A
正确答案：A
备注：作答正确，用于巩固知识点
考点：Cache的基本概念与功能

正确答案：B、C、D
备注：作答正确，用于巩固知识点
考点：总线/接口的传输方式

题目17

题干：在计算机硬件中，下列哪些设备属于外部设备？（ ）
A. CPU
B. 硬盘驱动器
C. 显示器
D. 键盘
你的答案：C、D
正确答案：B、C、D
错误原因：漏选B。CPU属于内部核心部件，硬盘驱动器（外存）、显示器（输出设备）、键盘（输入设备）均属于外部设备
考点：计算机硬件的分类
解题技巧：外部设备=输入设备+输出设备+外存设备；内部设备=CPU、内存、主板等

题目18

题干：下列哪些选项是计算机中使用高速缓存（Cache）的主要原因？（ ）
A. 解决CPU和主存储器之间的速度不匹配问题
B. 提高数据访问的局部性
C. 减少CPU的功耗
D. 增加系统的存储容量
你的答案：A、B、C
正确答案：A、B
错误原因：错选C。Cache的核心目的是解决CPU与主存速度差、利用数据局部性提升访问效率；它不减少CPU功耗，也不增加存储容量，C、D错误
考点：Cache的设计目的
解题技巧：Cache的本质是“用小容量高速存储，缓解CPU与主存的速度鸿沟”，与功耗、容量无关

题目19

你的答案：A、C、D
正确答案：A、B、C

解题技巧：HDMI可传音视频，显卡可独立/集成，多显卡可共存；显示模式有复制/扩展/仅第二屏幕等
