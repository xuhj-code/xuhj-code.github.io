计算机网络 - 子网划分十进制二进制速查表

发表于 2026-01-20 更新于 2026-03-08 分类于 专业知识 ， 计算机网络

本文聚焦事业单位计算机考试中子网划分的高频考点，系统整理十进制与二进制的核心对应关系、速查表、换算技巧及易错点提示，内容精准适配考试大纲，可直接用于备考复习。

考点提示：可用主机数公式 $2^n-2$（减2为网络地址和广播地址）是必考点，需熟练记忆。

三、 考试高频换算技巧

1. 子网掩码 → 块大小

步骤：① 找到二进制中最右边的1对应的十进制值；② 块大小 = $2^{\text{右边0的位数}}$。

示例：掩码224（11100000）→ 右边5个0 → 块大小 = $2^5=32$（事业单位常考）。

2. 块大小 → 子网掩码

步骤：① 块大小 = $2^n$ → 主机位为$n$位；② 子网掩码二进制 = 32-$n$个1 + $n$个0 → 转十进制。

示例：块大小64 = $2^6$ → 主机位6位 → 二进制11000000 → 十进制192。

3. 二进制 → 十进制（组合法）

核心：将二进制拆分为2的幂次之和，对应权重相加（记忆口诀：1、2、4、8、16、32、64、128）。

示例：11010000 → 128+64+16=208；10110000 → 128+32+16=176。

四、 事业单位考试易错点提示

  • 可用主机数必减2，忽略网络地址和广播地址会直接丢分；

  • CIDR前缀与子网掩码对应关系：/24对应255.255.255.0，/27对应255.255.255.224（高频对应）；

  • VLSM划分需遵循“先大后小”原则，先分配大子网，再拆分剩余地址给小子网，避免地址浪费。

五、高频速算技巧（非255/0八位组专属）

✅ 核心原则：仅对子网掩码最后一个非255的八位组计算，其余八位组直接复制/置0/置255(子网掩码为 255、0、252、248、240、224、192、128的常见情况)
✅ 3个核心公式

  1. 块大小（子网步长）= 256 - 子网掩码目标八位组数值
  2. 子网地址目标八位组 =（IP目标八位组 ÷ 块大小）取整 × 块大小
  3. 广播地址目标八位组 = 子网地址目标八位组 + 块大小 - 1
    ✅ 主机范围速推
    最小可用主机=子网地址+1 | 最大可用主机=广播地址-1
    ✅ 速算步骤（10秒搞定）
  4. 定位掩码最后一个非255八位组，计算块大小；
  5. 算子网/广播地址目标位，其余位直接复制；
  6. 首尾各+1/-1得主机范围。
    ✅ 实例：IP=172.16.5.68，掩码=255.255.255.224
    块大小=256-224=32 → 子网位=68÷32取整×32=64 → 广播位=64+32-1=95 → 结果：172.16.5.64/95，主机65-94

六、易错点避坑（考试高频丢分点）

❌ 二进制补0：转二进制必须8位一组，如10→00001010，非1010，否则位数错位
❌ 主机位全0/全1不可用：子网地址（全0）、广播地址（全1）排除在主机范围外
❌ 忽略变长子网掩码：A/B/C类默认掩码仅作参考，如B类172.16.5.68用/27掩码，网络位27位非16位
❌ 块大小错算：仅对最后一个非255八位组计算，掩码全255/0的八位组直接处理
❌ 取整错误：子网地址目标位计算为向下取整，如70÷32=2.1875，取整为2
❌ 掩码特征混淆：子网掩码二进制必是1连续在前、0连续在后，无1/0混合情况

七、IPv4子网计算高频考题

题型1（标准C类掩码，基础款）

已知：IP=192.168.3.56，子网掩码=255.255.255.0
求：子网地址、广播地址、可用主机范围
速算步骤：掩码全255/0，第四位0→子网位置0，广播位置255
答案：子网192.168.3.0 | 广播192.168.3.255 | 主机192.168.3.1~192.168.3.254

题型2（C类变长子网，/26掩码）

已知：IP=192.168.10.88，子网掩码=255.255.255.192
求：子网地址、广播地址、可用主机范围
速算步骤：块大小=256-192=64 → 子网位=88÷64取整×64=64 → 广播位=64+64-1=127
答案：子网192.168.10.64 | 广播192.168.10.127 | 主机89~126

题型3（B类变长子网，/27掩码）

已知：IP=172.20.18.95，子网掩码=255.255.255.224
求：子网地址、广播地址、可用主机范围
速算步骤：块大小=256-224=32 → 子网位=95÷32取整×32=64 → 广播位=64+32-1=95
答案：子网172.20.18.64 | 广播172.20.18.95 | 主机65~94

题型4（B类变长子网，/20掩码）

已知：IP=172.16.45.120，子网掩码=255.255.240.0
求：子网地址、广播地址、可用主机范围
速算步骤：掩码第三位非255，块大小=256-240=16 → 子网位=45÷16取整×16=32 → 广播位=32+16-1=47，第四位置255
答案：子网172.16.32.0 | 广播172.16.47.255 | 主机172.16.32.1~172.16.47.254

题型5（A类变长子网，考试偏难题）

已知：IP=10.8.66.200，子网掩码=255.255.192.0
求：子网地址、广播地址、可用主机范围
速算步骤：掩码第三位非255，块大小=256-192=64 → 子网位=66÷64取整×64=64 → 广播位=64+64-1=127，第四位置255
答案：子网10.8.64.0 | 广播10.8.127.255 | 主机10.8.64.1~10.8.127.254
